# DISINTEGRATION — Production Checklist

A single-page production checklist for *DISINTEGRATION*, an immersive performance piece. Saturday, June 20, 2026.

Open `index.html` in any browser. Progress is saved locally in the browser (localStorage). Tap any task to check it off. Tyler's tasks and Cansu's tasks are split into two columns per day.

## Publishing with GitHub Pages
1. Push this folder to a new GitHub repo.
2. In the repo: **Settings → Pages**.
3. Under "Build and deployment," set **Source: Deploy from a branch**, **Branch: main / root**.
4. Save. After a minute the live URL appears at the top of the Pages settings.
